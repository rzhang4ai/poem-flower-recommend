- 原典来源：《诗学含英》数字化文本 `诗学含英_simp_clean.txt`（1822行）
- 解析策略：
  - `## 标题` → 二级意象词条（如"初日"、"北斗"）

- 词典规模：232 个标题词条 + 10750 条子词，合计 10934 词
- 覆盖大类：天文类（73条）、地舆类（79条）、时令类（49条）、节序类（31条）

## step2e：情感分析

### 两阶段工作

#### Phase 1：FCCPSL 词典法（完成）
- 词典：FCCPSL（古诗词情感词典，OWL 格式）+ NTUSD
- 解析 `FCCPSL.owl` 提取情感词汇 → 3层情感体系：
  - **C3**（15维细粒度）：ease / joy / praise / like / faith / wish / peculiar / sorrow / miss / fear / guilt / criticize / anger / vexed / misgive
  - **C2**（5维）：pleasure / favour / surprise / sadness / disgust
  - **C1**（2维）：pos_score / neg_score

#### Phase 2：SikuRoBERTa 微调 FSPC（完成）
- 训练数据：FSPC v1.0（5000首古诗词，5类情感标注）
- 5类标签：负面 / 隐性负面 / 中性 / 隐性正面 / 正面
- 训练配置：3 epochs，batch=16，lr=2e-5，MPS，耗时 276s

**测试集指标（FSPC 500首）：**
| 指标 | 数值 |
|------|------|
| Accuracy | 50.20% |
| Macro-F1 | 47.58% |
| Weighted-F1 | 50.59% |

> 注：F1 约 50% 是古诗词情感分类正常水平，因该任务本身极具挑战性（大量隐性情感）。模型的主要价值是作为「情感特征提取器」，而非极性判断器。

**1075首推断置信度：** 平均 0.490，约 80% 集中在 0.40-0.80 区间。

## step2f：题材分类

### 工作内容
基于 **CCPD 数据集**（Wei et al., IEEE/ACM TASLP 2024，17103首标注），微调 SikuRoBERTa 进行 10类题材分类。

**10类标签**：思乡 / 咏物 / 山水 / 怀人 / 悼亡 / 咏史 / 田园 / 边塞战争 / 闺怨 / 送别

### 关键决策与修复记录

#### 数据解析 Bug 修复（本次对话核心工作之一）
- **问题**：`01_download_ccpd.py` 中 `line.split(",", 3)` 导致标签解析错误，`边塞战争=0首`、`咏史=0首`，`思乡`虚高至 57.7%
- **根本原因**：CCPD 原始数据用 `战争` 表示边塞战争、`怀古` 表示咏史；多标签用 `#` 分隔（如 `思乡#战争`）
- **修复方案**：
  1. 加入标签别名映射：`战争→边塞战争`，`怀古→咏史`
  2. 多标签取首个主标签（单标签任务）
- **修复效果**：`边塞战争` 1138首（6.7%），`咏史` 685首（4.0%），数据完整

#### 类别不均衡处理
- **问题**：思乡占 52.2%，送别仅 1.8%，直接导致模型全部预测思乡
- **方案**：双向均衡
  - 欠采样：思乡上限 2000首（52% → 19%）
  - 过采样：小类（咏物/悼亡/咏史/闺怨/送别）下限 800首
  - `label_smoothing_factor=0.1`（替代类别权重，避免 MPS NaN）
  - `max_grad_norm=1.0` + 分类头重新初始化

#### 训练结果（CCPD 测试集 1711首）
| 指标 | 数值 |
|------|------|
| **Accuracy** | **98.13%** |
| **Macro-F1** | **96.52%** |
| Weighted-F1 | 98.13% |
| 训练耗时 | 2690s（MPS，7 epochs，EarlyStopping） |

**各类 F1（最低两类）**：
- 边塞战争：0.9315（12/114 误分，主要混淆思乡，符合多标签本质）
- 送别：0.8750（测试集仅 31首，统计波动正常）
