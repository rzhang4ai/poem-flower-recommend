## 二、重点资源详解

### 2.1 🔴 images2poem — 600+ 意象词表（最直接可用）

**来源**：ACM MM 2018 论文《Images2Poem: Generating Chinese Poetry from Image Streams》
**地址**：https://github.com/liulixin29/images2poem

**内容**：
- 600+ 个在古诗中**频繁出现**的意象词汇（属性词 attribute words）
- 词汇来源：从大规模古诗语料中统计高频物象词，经人工筛选
- 涵盖：自然（月/风/雨/云/山/水）、植物（梅/兰/竹/菊/荷/柳）、
  动物（燕/雁/鹤/蝉/蝶）、器物（琴/剑/灯/舟）、情感动词（思/忆/归/别）

**与我们当前词典的对比**：

| 项目 | 我们当前 IMAGERY_VOCAB | images2poem 词表 |
|------|----------------------|-----------------|
| 规模 | ~90 词（3类各30词） | 600+ 词 |
| 来源 | 人工设计（Phase2方案） | 语料统计+人工筛选 |
| 覆盖度 | 基础核心词 | 更全面，含二级/三级意象词 |
| 可信度 | 需验证 | 学术发表，有数据支撑 |

**建议操作**：
```python
# 下载后替换/扩充 IMAGERY_VOCAB
# 在 visualize_embeddings.py 和 step2c_imagery.py 中使用
IMAGERY_VOCAB = load_images2poem_vocab("images2poem/attr_words.txt")
```

### 2.2 🔴 CCPD — 17K首诗主题+情感双标注（多标签分类基准）

**来源**：yuting-wei/CCPD（GitHub）
**地址**：https://github.com/yuting-wei/CCPD

**标注体系**：

#### 主题标签（10类）
| 标签 | 说明 | 与花卉推荐的关联 |
|------|------|----------------|
| 思乡 | 羁旅思乡 | 送别/思念场合 |
| 咏物 | 托物言志 | **直接相关**：咏花诗的主题标签 |
| 山水 | 山水田园 | 归隐/自然场合 |
| 怀人 | 思念故人 | 悼念/思念场合 |
| 悼亡 | 哀悼逝者 | 悼念场合 |
| 咏史 | 怀古感今 | 历史/文化场合 |
| 田园 | 田园隐逸 | 归隐/自然场合 |
| 边塞战争 | 征战边塞 | 励志/赠别场合 |
| 闺怨 | 闺中愁怨 | 爱情/思念场合 |
| 送别 | 别离送行 | **直接相关**：送别场合 |

#### 情感标签（12类）
`愁绪 / 哀伤 / 恐惧 / 惊讶 / 流泪 / 失意 / 思念 / 孤独 / 喜悦 / 怨恨 / 愤怒 / 想家`

**与本项目 8 维情感体系的映射**：
```
CCPD 12维 → 本项目 8维（joy/sorrow/longing/bold/elegant/serene/resentment/anxiety）
  喜悦 → joy
  愁绪+哀伤+失意 → sorrow
  思念+想家 → longing
  孤独 → serene（部分）
  怨恨+愤怒 → resentment
  恐惧+惊讶 → anxiety
  （bold/elegant：CCPD中无直接对应，需从咏物/山水主题推断）
```

### 2.3 🟡 南京大学物象库（2024，全唐诗 NER 标注）— 最接近需求

**来源**：刘懋霖、赵萌、王昊（南京大学信息管理学院），2024年发表
**论文**：《面向古诗词的物象库构建方法及其分布规律研究》
**期刊**：图书情报知识，2024年第43卷第393期

**核心价值**：
- 用 RoBERTa-BiLSTM-CRF 自动提取古诗意象（F1 最高 93.6%）
- 对《全唐诗》做了系统性意象 NER 标注
- 分类：普通名词（自然物象）、时间名词、地名
- **已迁移到宋词**，每首诗平均提取 4.5 个物象

**获取方式**：
- 联系作者索取标注数据（学术交流通道）
- 邮件：南京大学信息管理学院（查论文联系方式）
- 或：自行用论文方法复现（模型已描述清楚）

**可直接复现的方法**：
```python
# 论文描述的模型：RoBERTa-BiLSTM-CRF
# 标注类型：B-OBJ（物象起始）, I-OBJ（物象内部）, O（非物象）
# 训练数据：全唐诗子集（人工标注）
# 使用我们已有的 SikuRoBERTa 替换 RoBERTa 基础模型，
# 接 BiLSTM-CRF 层，在我们的200首样本上做推断
```

## 三、我们当前意象词典的问题与扩充建议

### 3.1 当前词典存在的不足

```python
# 当前 IMAGERY_VOCAB：~90词，分3类
NATURE_IMAGERY   = { "月","日","风","雪","霜","云","水","山" ... }  # ~40词
CULTURAL_IMAGERY = { "剑","琴","酒","烛","灯","镜","楼","舟" ... }  # ~25词
ACTION_IMAGERY   = { "飞","落","凋","零","断","寄","归","别" ... }  # ~25词
```

**主要不足**：
1. **规模偏小**：90词 vs images2poem 600词 vs 学界公认的500-1000个核心意象
2. **二级意象缺失**：如「东篱」「南浦」「折桂」等**文化复合意象**未收录
3. **花卉专项缺失**：作为花卉推荐项目，花卉的具体意象词（「疏影」「暗香」「凌寒」）未单独列类
4. **动词意象不足**：「折柳」「把酒」「望月」等动词性意象仅收录了单字版

### 3.2 扩充方案（按优先级）

**立即可做（用 images2poem 词表）**：
```python
# 1. 下载 images2poem attr_words.txt（600+词）
# 2. 与我们当前 IMAGERY_VOCAB 取并集
# 3. 人工去除明显不适合的词（如过于口语化的）
IMAGERY_VOCAB_V2 = IMAGERY_VOCAB | load_images2poem_vocab()
# 预计扩充到 500-600 词
```

**短期可做（增加花卉专项意象）**：
```python
FLOWER_IMAGERY = {
    # 梅花意象群（宋词常见）
    "疏影","暗香","凌寒","傲骨","铁骨","玉骨","孤高",
    # 荷花意象群
    "出淤泥","碧叶","晓露","芙蕖","芙蓉",
    # 菊花意象群
    "东篱","陶潜","晚节","黄花","傲霜",
    # 牡丹意象群
    "国色","天香","倾城","骄艳","帝城",
    # 桃花意象群
    "灼灼","夭夭","烂漫","桃源","避世",
}
```

**中期可做（引入南大物象库方法论）**：
- 基于我们的200首样本，用 SikuRoBERTa + BiLSTM-CRF 做意象 NER
- 与词典方法对比，取并集作为最终意象词表

## 四、建议采纳的意象分类体系

综合学界研究，建议将意象分为 **6 类**（比当前3类更细）：

| 类别 | 代表词 | 来源 | 说明 |
|------|--------|------|------|
| **天象时令** | 月、日、风、雪、霜、雨、露、云、春、秋 | 最高频，所有研究均收录 | 与时间、情感、节气直接相关 |
| **山水地理** | 山、水、江、湖、关山、阳关、玉门、长亭 | CCPD/PKG均有收录 | 含自然景观和文化地理意象 |
| **植物花卉** | 梅、兰、竹、菊、荷、柳、桃、松、桂、芳草 | images2poem重点 | 直接关联花卉推荐主题 |
| **动物禽鸟** | 雁、燕、鹤、蝉、蝶、鱼、鸿、杜鹃 | 所有研究均收录 | 鸿雁=书信，杜鹃=思乡（固定文化意象）|
| **器物人文** | 剑、琴、酒、舟、灯、镜、书、笛、烛 | PKG知识图谱 | 文人生活器物，承载情感 |
| **行为情感** | 飞、落、归、别、折柳、把酒、望月、登高 | 南大物象库 | 动态意象，在特定语境中触发情感 |
